    WORKER MODULE (worker.py):
    - Module-level function run_combo (NOT inside if __name__ == "__main__" or any closure).
    - Signature: run_combo(*, combo_dict: dict, fold_boundaries: list[dict], db_path: str, run_id: str, symbol: str, timeframe: str, seed: int, shard_dir: str, param_hash_str: str) -> list[dict]
    - Open read-only connection: conn = duckdb.connect(db_path, read_only=True)
    - For each fold in fold_boundaries:
        a. Parse fold["is_start"] and fold["oos_start"] with datetime.fromisoformat()
        b. Determine warmup_start: compute bars needed before is_start for indicator warmup. Fetch warmup + IS bars with: SELECT ... FROM bars WHERE symbol=? AND timeframe=? AND ts_utc >= ? AND ts_utc < ? ORDER BY ts_utc ASC (parameterized). warmup start = is_start minus (warmup_n * 1 minute * 1.5 as buffer). warmup_n = strategy.warmup_bars() after constructing strategy.
        c. Construct ORBConfig(**{k: v for k, v in combo_dict.items() if k in _ORBCONFIG_FIELDS}). _ORBCONFIG_FIELDS is a module-level frozenset computed at import time via dataclasses.fields(ORBConfig).
        d. Build Bar objects from DataFrame rows (same pattern as run_backtest.py lines 226-239).
        e. Slice warmup bars (ts_utc < is_start) and IS bars (is_start <= ts_utc < oos_start) from the fetched DataFrame.
        f. Prime strategy with warmup bars: call strategy._push_bar(bar) for each warmup bar WITHOUT calling on_bar.
        g. Run IS backtest: asyncio.run(engine.run(run_id=new_run_id(), bars=is_bars, strategy=strategy, risk_manager=PassThroughRiskManager(RiskConfig()), executor=PaperExecutor(symbol), seed=seed, init_cash=10_000.0))
        h. Reset strategy for OOS: construct fresh ORBStrategy(config), prime with warmup + IS bars (strategy.is_warm() should be True after warmup_n bars).
        i. Fetch OOS bars (oos_start <= ts_utc < oos_end exclusive). Run OOS backtest same way.
        j. Write IS equity curve Parquet to shard_dir/worker_{param_hash_str[:12]}_fold{fold_idx}_is.parquet using write_equity_parquet.
        k. Write OOS equity curve Parquet similarly.
        l. Compute edge_ratio: is_sharpe / oos_sharpe if oos_sharpe and abs(oos_sharpe) > 1e-9 else None.
        m. Append result dict: fold_idx, param_hash (param_hash_str), opening_range_minutes, atr_stop_mult, r_target, is_sharpe, oos_sharpe, is_return, oos_return, edge_ratio, equity_curve_path (OOS path as str), git_sha (call git_sha()), data_hash (call data_hash(full_df_for_fold)), seed.
    - conn.close() in finally block.
    - Return list of result dicts (one per fold).
    - Import guard at top: DO NOT import api, tv_bridge. Add a comment: "# D-07: workers import only trading-core"
