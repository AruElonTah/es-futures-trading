| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-08-06 | Information disclosure (secrets in logs) | CI job output | mitigate | Workflow defines no `secrets.*` usage and no `env` secrets; replay/backtest tests use synthetic fixtures only — nothing sensitive can land in logs |
| T-08-07 | Elevation (workflow injection) | workflow triggers | accept | Triggers limited to `push`/`pull_request` on master/main; no `pull_request_target`, no `workflow_run`; default `GITHUB_TOKEN` permissions are read-only for test jobs |
</threat_model>
