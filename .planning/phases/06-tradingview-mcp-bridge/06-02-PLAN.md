| Next.js client → POST /tv/focus | Untrusted request body crosses here; symbol must be allowlisted, date must be a valid ISO date |

| TVBridge → MCP server (stdio) | TVBridge constructs args dicts that go into the stdio JSON-RPC stream; the MCP server itself is trusted but the chart engine can be confused by garbage input |
| MCP server stderr → _stderr_capture (StringIO) | Server-side error messages may contain session URLs, tokens, or internal paths — must not be persisted in this plan |
