# Optimization Run ADR — ORB v1 Initial Grid Search (SPY 1m, Jan–Mar 2024)

**Date:** 2026-05-17
**Status:** accepted
**Author:** Nathaniel Greene

---

## Context

First optimization run against the ORB v1 strategy to establish a baseline parameter set
and verify the end-to-end optimization pipeline (grid search → walk-forward → leaderboard).
Uses SPY 1m bars as the working symbol (ES proxy per CLAUDE.md Phase 0 decision).

---

## Decision

Run a grid + walk-forward optimization with the following parameters:

### Required Fields

| Field | Value | Notes |
|-------|-------|-------|
| `is_oos_split` | IS=1m OOS=1m rolling step=1m | Reduced window for initial baseline with 3-month bar window |
| `optspace_path` | config/strategies/orb.optspace.yaml | 5×5×5 = 125 combos |
| `objective` | oos_sharpe | OPT-07 requirement |
| `seed` | 42 | Reproducibility |

### Walk-Forward Configuration

- **IS window:** 1 month (~21 trading days)
- **OOS window:** 1 month (~21 trading days)
- **Step:** 1 month rolling
- **Expected fold count:** 2 folds over a 3-month bar window

### Data Range

- **Symbol:** SPY
- **Timeframe:** 1m
- **From:** 2024-01-02
- **To:** 2024-04-01
- **Holdout guard:** no — data ends well before the 6-month holdout barrier (2025-11-17)

---

## Rationale

3-month window (Jan–Mar 2024) is the minimum viable range for a 2-fold IS=1m/OOS=1m
walk-forward. This run establishes a known-good baseline before expanding to longer
windows. The 5×5×5 = 125 combo grid tests the full opening_range_minutes ×
atr_stop_mult × r_target parameter space coarsely before any fine-grid refinement (D-03).

---

## Expected Outcomes

- OOS Sharpe > 0 for at least some parameter combos (directional signal present)
- IS/OOS edge ratio < 2.0 for leading candidates (no obvious overfitting)
- `opt_results` table populated with 125 × 2 = 250 rows

---

## Consequences

- ADR hash will be written to every `opt_runs` row for forensic provenance.
- Results are immutable once written — do not re-use a `run_id`.
- No holdout burn consumed (data range avoids the last 6 months).
